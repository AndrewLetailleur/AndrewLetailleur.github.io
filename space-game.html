A Dummy Test Phaser Project

A basic working space shooter game, featuring a mix of Space Invaders and Galaxian-lite enemies, using Phaser 2

Game Instructions (Keys):
Press Left/Right Arrow keys to move.
press Spacebar to fire.
Press P to pause game.

Game Instructions (Mouse):
Move cursor left/right to move Spaceship.
Click Left Mouse Button to fire.
